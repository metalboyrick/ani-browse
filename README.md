# AniBrowser

This is a simple web app that can browse animes with the AniList API.

## Environment Setup

Dependencies installation:
```
npm install
```

Run development server:
```
npm run dev
```

Build for production and test:
```
npm run build
npm run start
```
The application would be served in ```localhost:3000```.
